## Каталог пользователей

Простое одностраничное приложение на **React + TypeScript** для просмотра списка пользователей из публичного API DummyJSON (https://dummyjson.com/docs/users).

### Возможности

- **Список пользователей**: отображение имени, ника, email, аватара и краткой информации.
- **Поиск по имени**: фильтрация пользователей по имени/фамилии.
- **Пагинация**: переключение страниц с использованием `total`, `limit`, `skip` из API.

### Технологии

- **React 19**
- **TypeScript**
- CSS без UI-фреймворков

### Запуск

```bash
npm install
npm start
```
